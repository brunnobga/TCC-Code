Para realizar uma avaliação objetiva, basta selecionar dois vídeos a serem comparados. Eles devem ser necessariamente do tipo .yuv.

Clique em 'Adicionar' para acumular a tarefa na fila de tarefas.

Após criar todas as tarefas desejadas, clique em 'Iniciar' para executar cada uma delas. Ao final de cada execução, a gravação do valor no banco de dados é indicada.

Para visualizar os resultados, acesse a janela Resultados na janela principal.
