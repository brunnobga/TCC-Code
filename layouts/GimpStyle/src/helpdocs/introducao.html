Introdução

A tela inicial do SASQV2 apresenta os seguintes componentes:

0. Login
- Oferece os campos para entrada de usuário e senha, permitindo o uso da aplicação.
1. Gerenciador de sessão
- Inicia a tela de gerenciamento de sessão, onde é possível executar sessões de avaliação subjetiva.
2. Ferramentas
- Inicia a tela de ferramentas, dando acesso às ferramentas de degradação e avaliação objetiva.
3. Gerenciador de dados
- Inicia o Gerenciador de dados, permitindo gerenciar usuários e vídeos da base de dados.
4. Visualizador de resultados
- Abre o visualizador gráfico de resultados
5. Configurações
- Abre um diálogo que permite configurar aspectos básicos do sistema.
